# 画像認識してRPA

画面上で任意のイメージを探してその場所の中心にマウスカーソルを移動させる。



必要なライブラリ

```python
pip install pyautogui
pip install pillow
pip install opencv-python
```



| ライブラリ    | 機能                            |
| ------------- | ------------------------------- |
| pyautogui     | OSにキー押下やクリックを送信    |
| pillow        | 画像処理                        |
| opencv-python | 画像処理 (イメージの曖昧検索用) |

[PyAutoGUI · PyPI](https://pypi.org/project/PyAutoGUI/)
